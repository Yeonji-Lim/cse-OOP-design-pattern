# 생성 패턴

## 객체 생성

일반적으로 생성자를 제공한다.

다음 같은 경우에는 변형이 필요하다.

- 생성 객체에 초기값 줄 때
- 생성할 클래스를 선택하려 할 때

생성자는 메소드? -> 문법 규칙이 있음

- 다른 메소드와 다르게 new를 사용해야 함
- 정의하지 않으면 자바의 경우에 디폴트로 매개변수 없는 생성자를 제공함
- 상속을 받은 경우 super를 꼭 제일 먼저 써야 함. 따라서 생성자를 정의하지 않으면 오류

-> 생성자는 규칙을 가진 메소드다.

여러 생성자가 필요한 경우에는 서로 협력이 필요할 수 있다.

슈퍼 클래스의 생성자와는 항상 협력한다.

객체 생성을 클래스 로직 밖으로 옮길 필요가 있다.
