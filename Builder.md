# 빌더 패턴

객체 생성 로직을 클래스 밖으로 옮길 필요가 있다.

-> 서로 연관된 복잡한 생성자를 가지고 있는 경우

역할을 분담하고 단일 책임을 가지게 하기 위해서

"점차적인 데이터" : 모든 데이터가 한꺼번에 제공되지 못한다.
