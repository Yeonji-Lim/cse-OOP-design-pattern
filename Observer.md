# Observer Pattern

객체 사이에 1대 n의 의존관계이며 

객체 상태의 변화가 다른 의존 객체(Observer)에 통지되고

자동으로 업데이트되게 해야하는 경우에 사용하는 패턴.

데이터의 변경이 발생했을 경우 상대클래스나 객체에 의존하지 않으면서 데이터 변경을 통보하고자 할 때 유용하다.

차량의 연료가 소진될 때까지 주행 가능 거리를 출력해야 하는 경우 등이 이런 경우에 포함된다.
