# 중재자 패턴 (Mediator Pattern)

조정자 패턴.

어떻게 객체들의 집합이 상호작용하는지 함축해놓은 객체

행위 패턴의 일종 : 프로그램의 실행 행위를 변경할 수 있다.

객체 간 통신은 중재자 객체 안에 함축된다.

객체들은 더이상 다른 객체와 서로 직접 통신하지 않으며 대신 중재자를 통해 통신한다.
